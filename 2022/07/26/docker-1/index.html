docker-1

docker-compose.yaml中可以通过指定build由文件夹构建

docker的volume有两种形式

  • 使用命名的数据卷
  • 直接指定主机上的路径来创建数据卷

docker hub

docker hub中会介绍docker的详细情况
例如对于postgresql容器，docker中介绍了初始化脚本的文件路径/docker-entrypoint-initdb.d
以及数据存储路径/var/lib/postgresql/data
知道这些路径可以协助我们在容器中对数据库进行初始化以及数据持久化
